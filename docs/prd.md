# PRD

## 1. 项目名称
Frontend Interview Copilot

## 2. 项目简介
这是一个面向前端实习求职者的 AI 面试陪练工具。用户输入岗位 JD 和简历内容后，系统自动生成一组前端面试题，并在用户作答后给出评分、优点、短板和下一步改进建议。

## 3. 目标用户
- 正在准备前端实习/校招的学生
- 希望进行模拟面试练习的人

## 4. 用户流程
1. 进入首页
2. 输入目标岗位 JD
3. 输入个人简历内容
4. 点击开始模拟面试
5. 系统生成 5 道前端题目
6. 用户逐题回答
7. 系统生成复盘报告

## 5. 页面列表
- 首页
- 输入页
- 面试页
- 报告页

## 6. 核心功能
### 6.1 输入 JD 和简历
用户通过文本框输入岗位描述和个人简历。

### 6.2 生成面试题
系统基于 JD 和简历生成 5 道前端面试题。

### 6.3 回答问题
用户逐题输入回答，系统记录内容。

### 6.4 生成复盘报告
系统根据全部回答生成：
- 总体评价
- 优势
- 薄弱点
- 改进建议

## 7. 非功能要求
- 页面简洁美观
- 响应式布局
- 有 loading / error / empty 状态
- 本地保存会话进度

## 8. V1 不做
- 登录注册
- 文件上传解析
- 语音输入输出
- 数据库存储
- 多轮复杂追问
