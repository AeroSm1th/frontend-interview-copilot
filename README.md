# Frontend Interview Copilot

一个小而美的 AI 前端面试陪练项目。

## 项目目标
用户输入目标岗位 JD 和自己的简历内容，系统生成前端面试题，用户完成作答后，AI 输出评分与复盘建议。

## V1 功能
- 输入岗位 JD
- 输入简历内容
- AI 生成 5 道前端面试题
- 用户逐题回答
- AI 输出复盘报告

## V1 不做
- 登录注册
- 数据库
- PDF 文件解析
- 语音面试
- 多模型切换
- 云端历史记录

## 技术栈
- Next.js
- TypeScript
- Tailwind CSS
- OpenAI API
- localStorage

## 开发目标
- 20 天内完成
- 每天约 2 小时
- 项目可上线，可写入简历
